Meus Projetos

Alguns projetos que desenvolvi ao longo da faculdade e de forma pessoal

Ĩbá — Sistema de Monitoramento Ambiental Comunitário

Plataforma fullstack para registro e monitoramento de ocorrências ambientais como queimadas, desmatamento e garimpo ilegal. Desenvolvida com Java 21 + Spring Boot no backend e HTML/CSS/JS puro no frontend, integrada ao Google Maps API. Inspirada na COP 30, nos povos indígenas guardiões do bioma e das minhas vivências — o nome significa "Mata" em Patxohã, língua do povo Pataxó.

HTML CSS JavaScript Java Spring Boot Docker
GitHub Deploy

PokéCenter Admin - Sistema de Gerenciamento de Pokémons

PokéCenter Admin é uma aplicação fullstack desenvolvida para gerenciar Pokémons com autenticação JWT, CRUD completo e atualização em tempo real via WebSocket. O sistema permite cadastrar, editar e deletar Pokémons, com regras de ownership — cada usuário só pode editar e deletar os próprios. A lista global é compartilhada entre todos os usuários autenticados e atualiza automaticamente sem precisar recarregar a página. O projeto aplica arquitetura em módulos com NestJS, ORM com Prisma, autenticação stateless, testes unitários com Jest e deploy em produção na Vercel e Railway.

Next.js TypeScript Nest.js Prisma JWT WebSocket Jest Docker Vercel Railway
GitHub Deploy

Sacola Cheia — Gerenciamento de Orçamentos para Hortifruti

Aplicação web fullstack com autenticação via Keycloak, permitindo gerenciar produtos, clientes, pedidos e entregas com controle de perfis (Gerente e Funcionário). Backend em Java 21 + Spring Boot com MySQL, frontend em Angular 18.
